# 📊 Power BI Dashboard

## Overview
This repository contains a **Power BI dashboard** designed to deliver clear, actionable insights through interactive visualizations.  
The dashboard background is kept **white** for a clean, professional look that emphasizes readability and clarity.

## 🎯 Objectives
- Present key performance indicators (KPIs) in a structured, easy-to-understand format.
- Enable interactive filtering and drill-down for deeper analysis.
- Showcase proficiency in **Power BI**, data visualization, and dashboard design.

## 🛠️ Tools & Technologies
- **Power BI** – dashboard creation, interactive visuals, slicers, and filters
- **Data Modeling** – relationships, measures, and calculated columns
- **DAX (Data Analysis Expressions)** – custom calculations and KPIs
- **Visualization Techniques** – charts, tables, cards, slicers

## 📌 Features
- Clean **white background** for professional presentation
- Interactive charts and KPIs
- Drill-down functionality for detailed insights
- Dynamic filters and slicers
- Optimized layout for readability
